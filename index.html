Twitch Highlighted Chat

This web page allows you to display only the highlighted messages of your Twitch chat.

Build for streamers who are afraid of missing these kinds of messages.

Enter below the name of the channel from which you want to isolate the highlighted messages.

Press "GO" and keep this window open.

Moderators can use command "!hlt <MESSAGE>" to push their message on website.
